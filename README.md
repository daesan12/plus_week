# **리팩토링 트러블슈팅**

https://velog.io/@diqkraud/리팩토링Trouble-Shooting
트러블슈팅 글

 유지보수성과 확장성을 개선하기 위한 리팩토링 기술을 적용한 사례를 작성. 데이터베이스 최적화, 트랜잭션 무결성 보장, N+1 문제 해결 등 다양한 문제를 다뤘다.
